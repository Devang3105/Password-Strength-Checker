# 🔐 Password Strength Checker

A Python-based Password Strength Checker with a Tkinter GUI.

## Features

- Checks minimum password length
- Checks uppercase letters
- Checks lowercase letters
- Checks numbers
- Checks special characters
- Calculates a password strength score
- Displays suggestions for improving weak passwords
- Graphical User Interface (Tkinter)
- Unit tests using Python's unittest module

- Tkinter

## How to Run

Run the application:

```bash
python main.py
```

Run the tests:

```bash
python test_password_checker.py
```
